Напиши функцию map(fn, array), которая принимает на вход функцию и массив, и обрабатывает каждый элемент массива этой функцией, возвращая новый массив. Пример: function square(x) { return x * x; } // возведение в квадрат console.log(map(square, [1, 2, 3, 4])); // [1, 4, 9, 16] console.log(map(square, [])); // [] Обрати внимание: функция не должна изменять переданный ей массив: var arr = [1, 2, 3]; console.log(map(square, arr)); // [1, 4, 9] console.log(arr); // [1, 2, 3]
Решение задание. Создайте обьект Unils, в котором будет метод map, который мы рассматривали на уроке, метод для генерации случайных чисел, и метод для возведения числа в квадрате и метод для генерации массива из чисел в диапазоне, введенном пользователем. Далее получите от пользователя два числла задающих диапазон, в котором будут числа нашего массива, генерируемого выше описанной функцией. Сгенерированный массив передайте в метод map обьекта Unils для возведения всех элеметов массива в крадрат. Новый массив с числами возведенными квадрат выведите в консоль.
